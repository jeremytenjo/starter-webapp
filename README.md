# Web app starter

A modern, highly scalable, offline-first foundation with the best developer experience.

## Usage

```sh
npm run app:dev
```

## What's included?

- React
- Vite
- MUI
- Storybook
- Firebase
- Vitest
- Playwright
- Prettier
- ESlint
